# AWS Linux Web Server Project 🚀

## 📌 Description
This project demonstrates deployment of a web server on AWS EC2 using Ubuntu Linux.

## ⚙️ Steps Performed
- Launched EC2 instance
- Connected via SSH / Instance Connect
- Installed Apache web server
- Hosted a custom HTML website

## 🌐 Output
Website hosted on AWS public IP

## 🛠️ Tools Used
- AWS EC2
- Ubuntu Linux
- Apache
